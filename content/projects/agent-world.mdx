Agent World is a 4,125-line interactive 2D virtual office that serves as the public showcase for the PRISM AI agent system. Visitors can explore the office using WASD keyboard controls, interact with 34 different AI agents at their desks, and see what each agent does in real time.

## What You Will See

- **34 AI agents** displayed at individual workstations across a virtual office floor plan
- **WASD navigation** for desktop exploration
- **Mobile card-grid fallback** for phone and tablet visitors
- **Visitor tour mode** that walks you through the entire agent fleet
- **Demo mode** showcasing agent capabilities
- **Live status indicators** for each agent

## Technical Details

- Single-file deployment with zero external dependencies
- Custom 2D rendering with canvas-based office layout
- Sanitized for public display with zero sensitive data in source
- Hosted on Hostinger VPS with Docker nginx:alpine and Traefik TLS
- Custom domain: agents.prismaiconsultants.com

## Why It Exists

Most companies talk about AI agents. This is proof that one person actually built and deployed 34 of them. Agent World makes the invisible visible - you can walk through the system and meet every agent.
