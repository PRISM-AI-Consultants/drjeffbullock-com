# Anti-Clone Rules - DrJeffBullock.com vs PRISM

These 14 rules ensure DrJeffBullock.com feels like a sibling to PRISM, not a copy. Every layout decision, component choice, and visual treatment must pass this checklist.

## The Rules

- [ ] **1. No centered hero with single CTA button** - Use asymmetric editorial layout instead. Text on one side, media or visual element on the other. The centered-hero-with-button is PRISM's signature pattern. DrJeffBullock.com leads with editorial composition.

- [ ] **2. No dropdown navigation menus** - Use flat horizontal nav with scroll on mobile. Dropdowns add complexity and hide content hierarchy. The personal site's IA is flat enough to not need them.

- [ ] **3. No services/pricing sections anywhere** - This is a personal thought-leadership platform, not a service catalog. No pricing tables, no "packages," no tier comparisons. If someone wants to hire Jeff, they go to PRISM.

- [ ] **4. No "How It Works" 3-step pattern** - The numbered-steps-with-icons layout is a consulting site cliche. DrJeffBullock.com explains through narrative, not numbered steps.

- [ ] **5. No testimonial carousel pattern** - No rotating quotes. If social proof appears, it should be integrated into content (inline quotes, case references) rather than sitting in a dedicated carousel section.

- [ ] **6. No same section ordering as any PRISM page** - Map out PRISM's page structures and deliberately reorder. If PRISM goes Hero > Features > CTA > Testimonials, DrJeffBullock.com should never follow that sequence.

- [ ] **7. No product/shop grid with pricing** - Use a library/catalog aesthetic instead. Content items (playbooks, frameworks, tools) are presented as a curated collection, not a storefront. Think library shelves, not shopping cart.

- [ ] **8. No PRISM logos, brand marks, or "PRISM" in the hero** - PRISM can be referenced in body content or an about section, but never in the hero area. The hero is Jeff's personal brand space.

- [ ] **9. No chatbot widget in bottom-right** - No floating chat bubble. If there is an interactive AI element, it should be integrated into the page layout as a first-class component, not a tacked-on widget.

- [ ] **10. No purple-primary buttons** - Use dark/stone primary buttons (#292524 background, #FAFAF9 text). Violet (#7C3AED) is accent-only: hover states, links, highlights, focus rings. Never the primary button fill.

- [ ] **11. No same footer column layout** - PRISM uses a 3-column footer. DrJeffBullock.com uses 4-column or asymmetric layout. Different grid, different content groupings.

- [ ] **12. No "Book a Call" as primary CTA** - Use exploration verbs instead: "Explore," "Read," "Listen," "Play," "Dive In." The personal site invites discovery, not sales conversion.

- [ ] **13. No same card shadow depth** - PRISM uses `rgba(0,0,0,.6)` which is heavy. DrJeffBullock.com maxes out at `rgba(0,0,0,.1)` for a lighter, more modern elevation feel.

- [ ] **14. Different heading weight distribution** - Heavier on DrJeffBullock.com (700-800 weight), more editorial. PRISM uses standard weights. The personal site should feel like a magazine, not a brochure.

## How to Use This Checklist

1. Before finalizing any page layout, run through all 14 rules.
2. If any rule is violated, redesign that section.
3. Check the rules again after responsive adjustments - mobile layouts can accidentally reintroduce patterns (e.g., stacking elements into a centered hero pattern on small screens).
4. When adding new sections or pages, check against PRISM's corresponding page to ensure no structural mirroring.

## Quick Reference: What to Use Instead

| PRISM Pattern | DrJeffBullock.com Alternative |
|---|---|
| Centered hero + CTA | Asymmetric editorial (text + media split) |
| Dropdown nav | Flat horizontal nav, scroll on mobile |
| Service/pricing grid | Library/catalog cards (no prices) |
| 3-step "How It Works" | Narrative explanation or visual story |
| Testimonial carousel | Inline quotes within content |
| Purple primary buttons | Stone/dark primary, violet accent only |
| 3-col footer | 4-col or asymmetric footer |
| "Book a Call" CTA | "Explore" / "Read" / "Listen" / "Play" |
| Heavy card shadows (.6) | Soft card shadows (.05-.1) |
| Standard heading weights | Heavy editorial weights (700-800) |
